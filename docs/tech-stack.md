# Tech Stack

| Layer | Technologies | Notes |
| ----- | ------------ | ----- |
| Framework | SvelteKit | Static site generation with `@sveltejs/adapter-static` |
| Language | TypeScript | Type safety throughout |
| Styling | CSS Custom Properties | No framework, theming via CSS variables |
| Build | Vite | Bundled with SvelteKit |
| Hosting | GitHub Pages | Static deployment via GitHub Actions |
| Audio | Web Audio API | Seamless looping for ambient sounds |
| Fonts | Google Fonts | Cormorant Garamond for Deep Thought's voice |
